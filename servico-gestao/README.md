# Modelo solid de uma api em Nest.js

## Api sobre gestao de servisos de uma companhia de telefonia



## Installation

```bash
$ npm install
$ npm install mysql2


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
